## APÓS A GERAÇÃO INICIAL

Use estes prompts de ajuste, um de cada vez:

```
# Se textos saírem em inglês:
"Traduza todos os textos do site para português brasileiro"

# Se tipografia não estiver correta:
"Use Cormorant Garamond em todos os títulos e DM Sans no corpo.
Pesos apenas light e regular — sem negrito pesado"

# Se layout mobile estiver quebrado:
"Revise o layout mobile — cards devem empilhar em 1 coluna,
header deve ter menu hamburguer, hero deve funcionar em 375px"

# Se hero precisar de ajuste:
"O hero deve ter fundo escuro com overlay suave, texto centralizado,
headline grande em Cormorant Garamond e botão CTA outlined"
```
